# **App Name**: VedicVerse

## Core Features:

- Product Catalog: Display clothing items with details like name, description, price, and available sizes.
- Admin Product Management: Admin panel to add, edit, and remove products from the catalog.
- Scheduled Launches: Set product launch dates (e.g., stock available from November 10-15). After this period, the product is automatically removed from the main catalog, but visible in the admin panel.
- Upcoming Launches Display: Display the date of the next upcoming launch on the website. This is controlled in the admin panel.
- User Authentication: Google Login integration for user account creation and management. This facilitates saving their account details and order activity.
- Order Management: When a customer places an order, their details and the product details are sent to the admin panel for processing and tracking.
- AI Assisted Product Descriptions: Use generative AI to suggest marketing text when admin panel creates the new product. An LLM 'tool' can consider elements like category and garment type to compose a unique sales-oriented summary.
